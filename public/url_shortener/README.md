# url_shortener

Shorten URLs into simple redirect codes.

**Stack used:** Svelte-Kit, TailwindCSS, PostgreSQL

## setup

* Install the required packages using the package manager of your choice (`npm`, `yarn`, `pnpm`, etc...)
```
yarn
```
* build the project and run it
```sh
yarn build
node server.js
```
* (or) run directly
```sh
yarn dev
```
